概述：

卡当是特别容易上手的游戏。每人先发2张牌，你可以要第三张牌，如果第三张牌的点数在两张牌之间，则赢得比赛，获得底池

赢输情况：

普通卡中

卡中顺子

卡中豹子

卡中同花

没卡中

没卡中

无法卡

具体规则：

牌库：1副牌，52张，不带大小王；

人数：每桌2至6人游戏；

底注：每局发底牌之前，所有人需要先下1倍基数的底注。

盲注：坐下游戏的第一局需向桌面投放一定的数量的积分，分为以下两种：

1、普通新手盲注：当桌上筹码清空，新加入比赛

2、2倍普通前注：当桌面上筹码低于5倍基数时

宝藏：每局游戏在投注底注的时候，会有一定的数额进入彩池。

下注：每人发两张底牌（自己直接可以看到这两张牌），从Dealer位置的下家开始，逆时针轮流决定下注或弃牌。 

Dealer位置每副牌逆时针依次轮换。

翻牌：决定下注，并获得翻牌的过程，称为：翻牌。翻牌的结果有可能是：卡中、卡飞、撞柱、卡当、卡豹子、卡同花。

底池：一个游戏桌建立之后，玩家所有的底注和下注都进入底池，底池与游戏桌绑定，若所有玩家都离开，底池中还有剩余筹码，则当另外的玩家再参与到此桌游戏时，桌面筹码自动清空并进入彩池。

注：当桌面还有剩余筹码，当桌面只剩一个玩家，而其他玩家都离开时，桌面筹码将全部分配给最后一个玩家；但需要有其他玩家进入时，筹码才会自动分配！

挖宝：比赛场地的彩池会随选手的参与度而逐渐增加，满足下述4种条件即可挖宝成功。

普通四条：若底牌为对子，并且翻牌中豹子，则进入挖宝藏的下一个环节：在剩余牌堆中抽出1张牌，若形成四条则中2%彩金。

A四条/J四条：若挖宝中得的是A或者J的四条，将会获得10%彩金。

同花顺：若底牌为相同花色，数字之差为2~4，并且卡当卡中同花色的牌成为三张的同花牌，则进入挖宝环节：在剩余牌堆中抽出一张牌，满足条件后方可再抽一张牌，若形成5张的同花顺，则中25%的彩金；若形成4张同花顺，则中2%的彩金。

游戏结束：1、某玩家将底池赢空；2、所有玩家均完成弃牌或下注。
